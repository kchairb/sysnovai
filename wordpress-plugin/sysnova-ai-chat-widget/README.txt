Sysnova AI Chat Widget
======================

Quick install
-------------
1. Upload the plugin ZIP in WordPress: Plugins -> Add New -> Upload Plugin.
2. Activate "Sysnova AI Chat Widget".
3. Go to Settings -> Sysnova AI Widget.
4. Set API URL to your deployed endpoint, for example:
   https://api.sysnova.ai/api/public/chat
5. Set Stream URL:
   https://api.sysnova.ai/api/public/chat/stream
5. Save.

Notes
-----
- Domain is auto-detected from your WordPress site URL.
- Language can be set to: fr, ar, darija, en.
- Streaming can be enabled/disabled in plugin settings.
- Lead capture can be enabled/disabled in plugin settings.
- Ensure your domain is registered in Sysnova backend allowed domains.
